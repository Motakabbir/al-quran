## Getting Started

### Mobile App

```bash
cd mobile
npm install
npm start
```

The mobile app can be run on:
- iOS simulator
- Android emulator 
- Physical device via Expo Go app

### Web App

```bash
cd website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the web app.
